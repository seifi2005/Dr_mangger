# راهنمای استفاده از داده‌های تست

## نصب داده‌های تست

برای اضافه کردن داده‌های تست به دیتابیس، از دستور زیر استفاده کنید:

```bash
mysql -u root -p medical_system < database_test_data.sql
```

یا در phpMyAdmin:
1. دیتابیس `medical_system` را انتخاب کنید
2. به تب "Import" بروید
3. فایل `database_test_data.sql` را انتخاب کنید
4. روی دکمه "Go" کلیک کنید

## داده‌های اضافه شده

این اسکریپت SQL موارد زیر را اضافه می‌کند:

- **100 رشته پزشکی**: شامل رشته‌های مختلف پزشکی
- **500 کاربر**: با نقش‌های مختلف (system-admin, operator, acceptor, service-provider, support)
- **500 پزشک**: با اطلاعات کامل شامل تخصص، شماره نظام پزشکی، آدرس و مختصات جغرافیایی
- **1000 مرکز درمانی**: شامل بیمارستان، درمانگاه و مرکز درمانی
- **100 داروخانه**: با لوکیشن روی نقشه (مختصات جغرافیایی در محدوده ایران)

## استفاده از تاریخ شمسی

در view ها می‌توانید از تابع `toPersianDate()` استفاده کنید:

```php
<?php echo toPersianDate($date); ?> // خروجی: 1403/05/15
<?php echo toPersianDate($date, 'Y/m/d'); ?> // فرمت سفارشی
<?php echo toPersianDateWithMonth($date); ?> // خروجی: 15 مرداد 1403
```

## نکات مهم

- کدهای ملی و شماره پروانه‌ها به صورت تصادفی و یکتا تولید می‌شوند
- لوکیشن‌های داروخانه‌ها در محدوده جغرافیایی ایران قرار دارند
- تاریخ‌های تولد پزشکان بین 25 تا 75 سال پیش است
- وضعیت اکثر رکوردها "فعال" است (90% فعال، 10% غیرفعال)

## حذف داده‌های تست

اگر می‌خواهید داده‌های تست را حذف کنید، می‌توانید از دستورات زیر استفاده کنید:

```sql
TRUNCATE TABLE doctors;
TRUNCATE TABLE users;
TRUNCATE TABLE medical_centers;
TRUNCATE TABLE medical_specialties;
TRUNCATE TABLE pharmacies;
```

**توجه**: این دستورات تمام داده‌های جداول را حذف می‌کنند. قبل از اجرا مطمئن شوید که داده‌های مهمی ندارید.
